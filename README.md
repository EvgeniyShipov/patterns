Паттерны проектирования
==============

Проект представляет из себя сборник паттернов проектирования, 
на основе книги "Приемы объектно-ориентированного проектирования. Паттерны проектирования." 
Гамма Э., Хелм Р., Джонсон Р., Влиссидес Дж.

Порождающие паттерны
-------------------------
 - Абстрактная фабрика (Abstract Factory)
 - Строитель (Builder)
 - Фабричный метод (Factory Method)
 - Прототип (Prototype)
 - Одиночка (Singleton)
 
Структурные паттерны
-------------------------
  - Адаптер (Adapter)
  - Мост (Bridge)
  - Компоновщик (Composite)
  - Декоратор (Decorator)
  - Фасад (Facade)
  - Приспособленец (Flyweight)
  - Заместитель (Proxy)
 
Поведенческие паттерны
-------------------------
 - Цепочка обязанностей (Chain of Responsibility)
 - Команда (Command)
 - Интерпритатор (Interpreter)
 - Итератор (Iterator)
 - Посредник (Mediator)
 - Хранитель (Memento)
 - Наблюдатель (Observel)
 - Состояние (State)
 - Стратегия (Strategy)
 - Шаблонный метод (Template Method)
 - Посетитель (Visitor)
